# Simple Ecommerce

### Installation ###

* git clone https://github.com/rajaslucky89/SehatQ.git
* Run pod install

### Dependencies ###

* Swift 4.2
* Moya
* Alamofire
* RxSwift
* ObjectMapper
* RealmSwift
* KingFisher
* IQKeyboardManager
* Facebook SDK
* Google SDK

### Architecture ###

* MVVM + Interactor
